Частые вопросы

знать заранее

Нужна ли физическая подготовка?
+

Нет. Программа подойдёт женщинам с любым уровнем физической формы. Йога адаптируется под каждую участницу. Треккинг на Горелый требует базовой выносливости, Андрей ведёт группу в комфортном темпе.

Можно ли взять только один тур?
+

Да. Каждый из четырёх ключей — самостоятельный однодневный тур. Выберите один, два или пройдите полный цикл. Полный путь даёт синергию всех четырёх стихий.

Что взять с собой?
+

Удобная одежда для движения, слоёная одежда для тундры, непромокаемая куртка, треккинговая обувь. Коврик для йоги предоставляется. Подробный список высылается после бронирования.

Сколько человек в группе?
+

Группа не более 6–8 участниц. Маленький размер группы даёт индивидуальное внимание Андрея и создаёт безопасное пространство для трансформации.
